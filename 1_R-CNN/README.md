## 1. 프로젝트 정의
- 2 Stage 모델 시초가 되는 R-CNN 모델을 구현하고 연구한다.</br>
- 근간이 되는 개념을 이해하기 위하여 기초 학습이 주목적입니다.
## 2. 데이터
- DeepFashion2 데이터를 사용했습니다</br>
- 약 5일 간 프로젝트 기간이 주어져 데이터 양을 줄였습니다//

## 3. 진행과정
- R-CNN 원문 논문을 읽고 학습을 먼저 하였다.
- 기간이 짧았기 떄문에 데이터를 전처리 하고 난 후 작은 사이즈로 줄였다. 

## 4. 모델 오류 원인과 해결
- 코드로 구현하지 못함.

## 5. 결과
- 학습을 통해 R-CNN 의 원리와 구조를 파악했지만 모델 구현에 실패했다.

## 5. 한계점
- 논문을 읽고 원리를 파악했지만, 이 과정을 코드로 구현하는 방법을 알지 못했다. 
- 데이터가 있지만 어떻게 사용하는 지 몰라 시간을 많이 소비하였다. 
- 데이터 전처리와 사이즈 축소하는 과정에서 부족한 코딩실력이 드러났다.
